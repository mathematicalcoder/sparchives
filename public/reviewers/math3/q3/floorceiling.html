Floor and Ceiling Functions

Introduction

The floor and ceiling functions are the most common examples of piecewise-defined functions.

The floor function \(\lfloor x\rfloor\) is the greatest integer less than or equal to \(x\), while the ceiling function \(\lceil x\rceil\) is the smallest integer greater than or equal to $x$.

Essentially, this means that in the floor function we round down to the nearest integer, while in the ceiling function we round up. (Be careful about rounding negative numbers!) Let's look at a few examples to understand how the functions work:

Number Floor Ceiling
$3$ $\lfloor3\rfloor=3$ $\lceil3\rceil=3$
$\pi$ $\lfloor\pi\rfloor=3$ $\lceil\pi\rceil=4$
$-5.2$ $\lfloor-5.2\rfloor=-6$ $\lceil-5.2\rceil=-5$

Properties and Graph

Of course, we can round any real number, so the domain of both functions is $\mathbb R$, and since the outputs are integers, the range is $\mathbb Z$.

Now, we mentioned earlier that the floor and ceiling functions are piecewise-defined functions, so let's write them using the standard notation we've been using for piecewise functions. The floor function would output $0$ if $0\le x\lt 1$, $1$ if $1\le x\lt2$, and so on, so we have: $$\lfloor x\rfloor=\begin{cases}\dots\\-1,&-1\le x\lt0\\0,&0\le x\lt1\\1,&1\le x\lt2\\\dots\end{cases}$$

We can do similarly for the ceiling function: $$\lceil x\rceil=\begin{cases}\dots\\0,&-1\lt x\le0\\1,&0\lt x\le1\\2,&1\lt x\le2\\\dots\end{cases}$$

Transformations

Watch this video to see the possible transformations of $\lfloor x\rfloor$ and $\lceil x\rceil$. We will use the forms $a\lfloor x-h\rfloor+k$ and $a\lceil x-h\rceil+k$.


As you can see, the transformations are just like those of the other functions we've seen so far:

  • The value of $a$ changes how far apart the pieces are.
  • The value of $h$ shifts the function $h$ units to the right.
  • The value of $k$ shifts the function $k$ units up.
